Rating

Rating components used for taking reviews from users and helps in providing others opinions and experience.

Read only Rating

Read only ratings are for users to see review on products or services you can't change the review. In order to implement rating just add .lg,.md,.sm class along with .rating class

Disabled ratings

For using disabled rating add .disabale_rating class.
