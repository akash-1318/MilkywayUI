Cards

Cards are built with as little markup and styles as possible, but still manage to deliver a ton of control and customization.

Vertical Cards with Badge

These are Vertical cards design with image, text, title and card shadow. To use the Card Component, put card-image, card-title and card-text, inside a div with class card and for the second card to make the name over image add class .card__name-over-img along with .card__name class and also add .title__over-img class along with .card__desc class.
