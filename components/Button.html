Buttons

Buttons allow users to take actions, and make choices, with a single tap. Use different classes created for different button styles to implement in your project.

Primary Buttons

Here are some Primary style buttons you can use by integrating classes for differnt styles like for primary button use .solid__primary along with .btn class.

Secondry Button Styles

Here are some Secondry style buttons you can use by integrating classes for differnt styles like for secondry button use .solid__secondry along with .btn class.

Different button size

To implement different sizes of button into your project just add class .lg-btn, .md-btn, .sm-btn for large, medium and small sizes buttons respectively.

Floating and Disabled Button

Here are the examples of disabled and floating button. To implement disabled button just add disabled keyboard in button element and add class .btn-disabled for disabled button style and to implement floating button just add .floating__btn class
