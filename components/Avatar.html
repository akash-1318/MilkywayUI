Avatar

Here are some different types of Avatar those can be used for showing profile pictures, from tables to dialog menus.

Image Avatar

This image Avatar component is typically used to display circular user profile pictures. You can use different size of Avatars by just including the classes like add .avatar class and for different font size add .avatar-lg-size, .avatar-md-size, .avatar-sm-size, .avatar-xs-size.

Text Avatar

Avatars containing simple characters can be created by passing a paragranph tag with class .lg__text-avatar, .md__text-avatar, .sm__text-avatar, .xs__text-avatar. for different font sizes.
